# Vue技巧

## 设置全局变量和函数

```js
// util.js
export default {
  install(Vue, options) {
    Vue.prototype.getData = function () {
      console.log('我是插件中的方法')
    }
  }
}

// main.js
import util from 'path/util.js'
Vue.use(util)

```
