##### 跳转下\上一个方法

ctrl + Shift + down\up

##### 格式化

Option + Command + L

##### 快速修复

Option + Enter

##### 查看错误信息

Command + F1

##### 定位到当前文件

Option + F1,1

##### 删除行

Command + del

##### 聚焦到上次的工具窗口

F12

##### 搜索目录结构，类似面包屑的查询

Command + F12

##### 下\上一个tab(改)

Command + ] 、Command + [

##### 打开项目(add)

ctrl + Command + Option + o

##### 打开最近项目

Shift + Command + Option + o

##### 文件内替换

Command + r

##### 全局替换

Command + Shift + r

##### 生成测试类

Command + Shift + t

##### 添加光标

Option + Shift + click
