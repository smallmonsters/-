#### Mac快捷键

##### 自动匹配相对应的语法结构，比如if，do-while，try-catch等结构

Command + Shift + Enter

##### 复制行/所选内容

Command + D

##### 选中一个标签内所有的内容

Option + up\down

##### 导入包,自动修正

Option + Enter

##### 代码上\下移

Command + Shift + up\down

##### 返回至上次浏览的位置

Command + Option + left/right

##### 左右切换tab

Command + Shift + []

##### 格式化

Option + Command + L
