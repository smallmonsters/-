链表（Linked List）是一种常见的线性数据结构，它由一系列节点组成，每个节点包含一个数据元素和一个指向下一个节点的引用（指针）。链表中的节点可以动态添加和删除，不需要像数组一样事先定义容量大小。链表可以用于各种数据结构和算法中，例如栈、队列、哈希表等。

需要注意的是，链表的访问速度相对于数组来说比较慢，因为链表中的每个节点需要占用额外的空间来存储指针信息。另外，由于链表是一种动态结构，也需要额外的空间来存储指针信息，因此空间开销也比数组大。

- 优点
  链表的优点是插入和删除操作效率高，因为它不需要像数组一样移动元素来完成这些操作。

- 缺点
  链表的缺点是访问操作效率较低，因为它不能像数组一样通过下标直接访问元素，而是需要从头节点开始遍历整个链表，直到找到需要访问的元素。

链表相比于数组，具有以下优点：

- 插入和删除操作快：由于链表是一种动态结构，可以很方便地进行节点的插入和删除操作。
- 空间利用率高：链表不需要预先分配固定大小的空间，而是根据实际需要动态分配空间，因此空间利用率比数组高。
- 可扩展性好：由于链表的动态特性，可以轻松地扩展链表的长度。

## 常见的链表类型

- 单链表
  单链表每个节点只有一个指向下一个节点的引用；

  ```java
    public class SinglyLinkedList<T> {
      // 使用了一个内部类 Node 来表示单向链表中的节点
      private class Node {
          T data;
          Node next;
          
          public Node(T data) {
              this.data = data;
          }
      }
      // 使用 head 指针指向单向链表的头部
      private Node head;
      private Node tail;

      
      public SinglyLinkedList() {
          this.head = null;
      }
      
      public boolean isEmpty() {
          return head == null;
      }
      // insertFirst() 方法用于在单向链表的头部插入一个新节点
      public void insertFirst(T data) {
          Node node = new Node(data);
          node.next = head;
          head = node;
      }
      public void insertFirst(T data) {
        Node node = new Node(data);
        if (isEmpty()) {
            tail = node;
        }
        node.next = head;
        head = node;
      }
      // insertLast() 方法用于在单向链表的尾部插入一个新节点
      public void insertLast(T data) {
          Node node = new Node(data);
          if (isEmpty()) {
              head = node;
          } else {
              Node current = head;
              while (current.next != null) {
                  current = current.next;
              }
              current.next = node;
          }
      }
      public void insertLast(T data) {
        Node node = new Node(data);
        if (isEmpty()) {
            head = node;
        } else {
            tail.next = node;
        }
        tail = node;
      }
      // deleteFirst() 方法用于删除单向链表的第一个节点
      public T deleteFirst() {
          if (isEmpty()) {
              throw new NoSuchElementException("List is empty");
          }
          T data = head.data;
          head = head.next;
          return data;
      }
      public T deleteFirst() {
        if (isEmpty()) {
            throw new NoSuchElementException("List is empty");
        }
        T data = head.data;
        if (head == tail) {
            tail = null;
        }
        head = head.next;
        return data;
      }
      // peekFirst() 方法用于查看单向链表的第一个节点
      public T peekFirst() {
          if (isEmpty()) {
              throw new NoSuchElementException("List is empty");
          }
          return head.data;
      }
      // size() 方法用于返回单向链表中节点的个数
      public int size() {
          int size = 0;
          Node current = head;
          while (current != null) {
              size++;
              current = current.next;
          }
          return size;
      }

  }

  ```

- 双向链表LinkedList
  每个节点既有一个指向下一个节点的引用，也有一个指向上一个节点的引用；

  ```java
    public class DoublyLinkedList<T> {
      // 使用了一个内部类 Node 来表示双向链表中的节点
      private class Node {
          T data;
          // 使用 head 和 tail 指针分别指向双向链表的头部和尾部
          Node prev;
          Node next;
          
          public Node(T data) {
              this.data = data;
          }
      }
      
      private Node head;
      private Node tail;
      
      public DoublyLinkedList() {
          this.head = null;
          this.tail = null;
      }
      
      public boolean isEmpty() {
          return head == null;
      }
      // insertFirst() 方法用于在双向链表的头部插入一个新节点
      public void insertFirst(T data) {
          Node node = new Node(data);
          if (isEmpty()) {
              tail = node;
          } else {
              head.prev = node;
          }
          node.next = head;
          head = node;
      }
      // insertLast() 方法用于在双向链表的尾部插入一个新节点
      public void insertLast(T data) {
          Node node = new Node(data);
          if (isEmpty()) {
              head = node;
          } else {
              tail.next = node;
              node.prev = tail;
          }
          tail = node;
      }
      // deleteFirst() 方法用于删除双向链表的第一个节点
      public T deleteFirst() {
          if (isEmpty()) {
              throw new NoSuchElementException("List is empty");
          }
          T data = head.data;
          if (head == tail) {
              head = null;
              tail = null;
          } else {
              head = head.next;
              head.prev = null;
          }
          return data;
      }
      // deleteLast() 方法用于删除双向链表的最后一个节点
      public T deleteLast() {
          if (isEmpty()) {
              throw new NoSuchElementException("List is empty");
          }
          T data = tail.data;
          if (head == tail) {
              head = null;
              tail = null;
          } else {
              tail = tail.prev;
              tail.next = null;
          }
          return data;
      }
      // peekFirst() 方法用于查看双向链表的第一个节点
      public T peekFirst() {
          if (isEmpty()) {
              throw new NoSuchElementException("List is empty");
          }
          return head.data;
      }
      // peekLast() 方法用于查看双向链表的最后一个节点，size() 方法用于返回双向链表中节点的个数
      public T peekLast() {
          if (isEmpty()) {
              throw new NoSuchElementException("List is empty");
          }
          return tail.data;
      }
      
      public int size() {
          int size = 0;
          Node current = head;
          while (current != null) {
              size++;
              current = current.next;
          }
          return size;
      }
  }

  ```

- 循环链表等。
  循环链表则是链表的最后一个节点指向链表的第一个节点，形成一个环。

  ```java
  public class CircularLinkedList<T> {
    private class Node {
        T data;
        Node next;
        
        public Node(T data) {
            this.data = data;
        }
    }
    //tail 指针指向循环链表的最后一个节点
    private Node tail;
    private int size;
    
    public CircularLinkedList() {
        this.tail = null;
        this.size = 0;
    }
    
    public boolean isEmpty() {
        return size == 0;
    }
    // 在循环链表的头部插入一个新节点
    public void insertFirst(T data) {
        Node node = new Node(data);
        if (isEmpty()) {
            tail = node;
            tail.next = tail;
        } else {
            node.next = tail.next;
            tail.next = node;
        }
        size++;
    }
    // 在循环链表的尾部插入一个新节点
    public void insertLast(T data) {
        insertFirst(data);
        tail = tail.next;
    }
    // 删除循环链表的第一个节点
    public T deleteFirst() {
        if (isEmpty()) {
            throw new NoSuchElementException("List is empty");
        }
        Node head = tail.next;
        if (head == tail) {
            tail = null;
        } else {
            tail.next = head.next;
        }
        size--;
        return head.data;
    }
    // 查看循环链表的第一个节点
    public T peekFirst() {
        if (isEmpty()) {
            throw new NoSuchElementException("List is empty");
        }
        return tail.next.data;
    }
    // 返回循环链表中节点的个数
    public int size() {
        return size;
    }
  }
  ```

## 链表获取中间元素

- 快慢指针法
  该方法需要定义两个指针，一个慢指针 slow，一个快指针 fast。快指针每次移动两个节点，慢指针每次移动一个节点。当快指针到达链表尾部时，慢指针指向的节点就是链表的中间节点

  ```java
  public T getMiddle() {
    if (isEmpty()) {
        throw new NoSuchElementException("List is empty");
    }
    Node slow = head;
    Node fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
    }
    return slow.data;
  }
  ```

- 计数法
  该方法需要先遍历一次链表，计算出链表的长度。然后再遍历一次链表，找到链表的中间节点。如果链表长度为奇数，中间节点是正中间的节点；如果链表长度为偶数，中间节点是靠近链表尾部的节点

  ```java
  public T getMiddle() {
    if (isEmpty()) {
        throw new NoSuchElementException("List is empty");
    }
    int size = 0;
    Node current = head;
    while (current != null) {
        size++;
        current = current.next;
    }
    int middleIndex = size / 2;
    current = head;
    for (int i = 0; i < middleIndex; i++) {
        current = current.next;
    }
    return current.data;
  }
  ```

## 链表的动态性

链表的动态性指的是链表的长度可以随时发生变化，可以动态地添加或删除元素。与数组不同，数组在创建时需要指定其长度，并且其长度是固定不变的，无法动态扩展或缩小。
